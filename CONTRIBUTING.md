# Contributing to Yuuki Chess Engine

Thank you for your interest in contributing! Here are a few guidelines to get started.

## How to Contribute
1.  Fork the repository.
2.  Create a new branch for your feature or bugfix (`git checkout -b feature/your-feature-name`).
3.  Make your changes.
4.  Ensure your code compiles and passes any existing tests.
5.  Commit your changes with a clear, descriptive message.
6.  Push your branch and open a Pull Request.

## Code Style
- Please follow the existing C++17 style used in the project.
- Use meaningful variable and function names.
- Comment your code where necessary.

## Reporting Issues
- Use the GitHub Issue tracker to report bugs.
- Provide as much detail as possible, including your OS and a description of the problem.
